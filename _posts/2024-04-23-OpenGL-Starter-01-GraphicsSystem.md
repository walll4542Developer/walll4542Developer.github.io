## 개요
그래픽스 시스템에 대해서 전체적으로 알아봅니다.

### 컴퓨터 그래픽스의 정의

컴퓨터 그래픽스라는 단어는 넓은 의미로 모호하게 사용되고 있습니다.
컴퓨터를 이용한 그래픽 디자인 또는 결과물
- 2D : 포토샵, 일러스트레이터 등
CG, 컴퓨터 애니메이션
- 3D : max, maya, blender 등
프로그래밍 관점
컴퓨터를 이용해서 이미지 / 애니메이션을 만들어내는 모든 기술


컴퓨터 그래픽스의 궁극적인 목표는 현실 세계를 시뮬레이션(Simulation)하는데 목적이 있습니다.
먼저 실제 또는 가상 물체를 시뮬레이션 합니다. 모델링 데이터
현실 세계의 빛의 동작을 시뮬레이션 합니다. 반사, 굴절, 회절
물리학 또는 광학등 과학의 시뮬레이션
인간의 시각 또는 카메라의 시뮬레이션

어떻게 시뮬레이션 할 것인가
물리학 / 광학 / 수학 방정식(Equation)을 프로그래밍하여 구현합니다.
다만 개인용 컴퓨터 사양에서 실시간(Realtime)으로 시뮬레이션 하기에는 컴퓨팅 파워가 부족하기 때문에 근사(Approximation) 기법을 사용합니다.

### 이미지 생성(Image synthesis)

현실세계의 시뮬레이션

기하학 
실세계 => 3차원 공간
실제 물체 => 오브젝트
카메라 => 뷰어

광학
고전 광학 이론을 가져와서 컴퓨터 상에서 광원, 반사, 굴절, 회절 등을 시뮬레이션 합니다.
인간 시각 시스템을 시뮬레이션 해야 되는데 시신경이 받아들인 정보를 뇌가 후처리를 거쳐
