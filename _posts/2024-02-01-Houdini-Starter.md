## 인터페이스 / 작업 환경 설정하기
많은 다른 DCC 툴과 같이 후디니는 인터페이스와 작업 환경을 커스텀 할 수 있습니다.
또한 이를 프리셋을 만들어 저장하는 기능을 제공합니다.

![FSR2](/assets/images/Docs/Houdini%20Starter/002.png){: .align-center}

예를 들어 위 이미지 처럼 작업 환경을 설정하고 저장하고자 한다면

![FSR2](/assets/images/Docs/Houdini%20Starter/006.png){: .align-center}

후디니 창에서 왼쪽 위 Build 버튼을 눌러줍시다. 

![FSR2](/assets/images/Docs/Houdini%20Starter/001.png){: .align-center}

위 이미지와 같은 메뉴가 나오는데요, Save Current Desktop As... 버튼을 누르면 지금 화면에 설정 되어있는 작업 환경을 프리셋으로 저장할 수 있습니다.

![FSR2](/assets/images/Docs/Houdini%20Starter/003.png){: .align-center}

저장할 프리셋의 이름을 지정해 줄 수 있습니다. 저는 MyShelf 라는 이름을 써서 저장했습니다.

![FSR2](/assets/images/Docs/Houdini%20Starter/004.png){: .align-center}

Save 버튼을 눌러주면 지정된 경로에 성공적으로 저장되었다는 알림창이 나옵니다.

![FSR2](/assets/images/Docs/Houdini%20Starter/005.png){: .align-center}

OK 버튼을 눌러주면 인터페이스가 성공적으로 저장되었음을 알 수 있습니다.
